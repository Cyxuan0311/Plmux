# 快捷键

plmux 中所有快捷键均以**前缀键**（默认 `Ctrl+B`）开头。先按前缀键，再按操作键。

## 前缀键

默认前缀为 `Ctrl+B`。可在 `config.json` 中修改：

```json
{
  "keys": {
    "prefix": "ctrl+a"
  }
}
```

支持的前缀值：`ctrl+b`、`ctrl+a`、`c-b`、`c-a`、`^b`、`^a`。

实现：[schema.py](../plmux/config/schema.py) | [prefix.py](../plmux/modes/prefix.py)

## 窗格管理

| 操作 | 默认绑定 | 说明 |
|------|----------|------|
| 垂直分割 | `%` 或 `v` | 将当前窗格左右分割 |
| 水平分割 | `"` 或 `s` | 将当前窗格上下分割 |
| 仅保留当前窗格 | `o` | 关闭其他窗格，仅保留当前窗格 |
| 缩放窗格 | `z` | 切换当前窗格全屏显示 |
| 焦点左移 | `h` | 将焦点移至上一个窗格 |
| 焦点右移 | `l` | 将焦点移至下一个窗格 |
| 焦点上移 | `k` | 将焦点移至上一个窗格 |
| 焦点下移 | `j` | 将焦点移至下一个窗格 |
| 焦点（方向键） | `←` `→` `↑` `↓` | 使用方向键移动焦点 |
| 向左调整大小 | `H` | 从左边缘缩小窗格 |
| 向右调整大小 | `L` | 向右扩展窗格 |
| 向上调整大小 | `K` | 从顶部缩小窗格 |
| 向下调整大小 | `J` | 向下扩展窗格 |

## 窗口管理

| 操作 | 默认绑定 | 说明 |
|------|----------|------|
| 新建窗口 | `c` | 创建新窗口 |
| 下一个窗口 | `n` | 切换到下一个窗口 |
| 上一个窗口 | `p` | 切换到上一个窗口 |
| 跳转窗口 0-9 | `0`–`9` | 按索引切换窗口 |
| 关闭窗口 | `&` | 关闭当前窗口（最后一个窗口时退出） |
| 循环布局 | `Space` | 循环切换布局模板 |

## 模式切换

| 操作 | 默认绑定 | 说明 |
|------|----------|------|
| 进入复制模式 | `[` | 进入文本选择模式 |
| 显示帮助 | `?` | 打开帮助覆盖层 |
| 分离会话 | `d` | 分离会话（会话在后台继续运行） |
| 进入命令行 | `:` | 打开 `:` 命令提示符 |

## 直接按键（无需前缀）

| 按键 | 操作 |
|------|------|
| `Ctrl+Q` | 强制退出 plmux |
| `Esc` | 直接透传给子程序（如 vim 退出插入模式） |

## 鼠标操作

plmux 通过 SGR 鼠标协议（DEC 模式 1006）支持鼠标交互。鼠标事件在启动时自动启用。

实现：[mouse_handler.py](../plmux/app/mouse_handler.py)

| 操作 | 说明 |
|------|------|
| 滚轮向上 | 向上滚动窗格内容（回滚缓冲区） |
| 滚轮向下 | 向下滚动窗格内容 |
| 左键点击窗格 | 切换焦点到被点击的窗格 |
| 左键点击边框 | 开始拖拽调整窗格大小 |
| 拖拽边框 | 按比例调整相邻窗格大小 |
| 子程序鼠标事件 | 子程序启用鼠标模式时自动转发（如 vim 鼠标模式、less 滚动） |

当子程序（如 vim 设置了 `set mouse=a`）启用鼠标模式时，plmux 会检测到并将所有鼠标事件直接转发给该程序，而非自行处理。

## 复制模式

复制模式使用类 Vim 的按键绑定进行文本导航和选择。

实现：[copy_mode.py](../plmux/modes/copy_mode.py)

| 按键 | 操作 |
|------|------|
| `h` / `←` | 光标左移 |
| `j` / `↓` | 光标下移 |
| `k` / `↑` | 光标上移 |
| `l` / `→` | 光标右移 |
| `w` | 前进一个单词 |
| `b` | 后退一个单词 |
| `0` / `Home` | 移至行首 |
| `$` / `End` | 移至行尾 |
| `g` | 移至第一行 |
| `G` | 移至最后一行 |
| `Ctrl+u` | 向上滚动半页 |
| `Ctrl+d` | 向下滚动半页 |
| `Space` | 开始/切换选择 |
| `Enter` | 复制选择到剪贴板并退出复制模式 |
| `v` | 切换字符/行选择模式 |
| `Esc` / `q` | 退出复制模式 |

## 命令行模式

按 `Prefix` 然后 `:` 进入命令模式。使用 `Tab` 进行自动补全。

实现：[commands.py](../plmux/input/commands.py)

| 命令 | 说明 |
|------|------|
| `:exit` | 强制退出（清除所有保存状态） |
| `:split`, `:sp` | 水平分割 |
| `:vsplit`, `:vsp`, `:vs` | 垂直分割 |
| `:only` | 仅保留当前窗格 |
| `:focus <n>` | 按索引聚焦窗格 |
| `:theme <name>` | 切换主题 |
| `:theme list` | 打开主题浏览器 |
| `:layout` | 打开布局浏览器 |
| `:layout <name>` | 应用指定布局模板 |
| `:web [port]` | 启动 Web 客户端（默认端口 9888） |
| `:webstop` | 停止 Web 客户端服务器 |
| `:ls` | 打开会话浏览器 |
| `:plugins` | 打开插件管理器 |
| `:reload`, `:source` | 重新加载配置并加载新启用的插件 |
| `:help` | 显示帮助覆盖层 |

## 自定义快捷键

可在 `config.json` 的 `keys.bindings` 中自定义快捷键：

```json
{
  "keys": {
    "prefix": "ctrl+b",
    "bindings": {
      "split-vertical": ["%", "v"],
      "split-horizontal": ["\"", "s"],
      "only-pane": ["o"],
      "next-window": ["n"],
      "prev-window": ["p"],
      "new-window": ["c"],
      "close-window": ["&"],
      "copy-mode": ["["],
      "cycle-layout": [" "],
      "help": ["?"],
      "detach": ["d"],
      "focus-left": ["h"],
      "focus-right": ["l"],
      "focus-up": ["k"],
      "focus-down": ["j"],
      "resize-left": ["H"],
      "resize-right": ["L"],
      "resize-up": ["K"],
      "resize-down": ["J"],
      "zoom": ["z"],
      "command-line": [":"]
    }
  }
}
```

每个操作映射到一个按键列表。第一个匹配的按键触发操作。可以为同一操作添加多个按键，也可以通过从列表中移除按键来删除绑定。

### 可用绑定操作

| 操作 | 说明 |
|------|------|
| `split-vertical` | 左右分割窗格 |
| `split-horizontal` | 上下分割窗格 |
| `only-pane` | 仅保留当前窗格 |
| `next-window` | 切换到下一个窗口 |
| `prev-window` | 切换到上一个窗口 |
| `new-window` | 创建新窗口 |
| `close-window` | 关闭当前窗口 |
| `copy-mode` | 进入复制模式 |
| `cycle-layout` | 循环布局模板 |
| `help` | 显示帮助覆盖层 |
| `detach` | 分离会话 |
| `focus-left` | 焦点移至上一个窗格 |
| `focus-right` | 焦点移至下一个窗格 |
| `focus-up` | 焦点移至上一个窗格 |
| `focus-down` | 焦点移至下一个窗格 |
| `resize-left` | 向左调整窗格大小 |
| `resize-right` | 向右调整窗格大小 |
| `resize-up` | 向上调整窗格大小 |
| `resize-down` | 向下调整窗格大小 |
| `zoom` | 切换窗格缩放 |
| `command-line` | 进入命令行模式 |

实现：[schema.py](../plmux/config/schema.py) | [prefix.py](../plmux/modes/prefix.py)
